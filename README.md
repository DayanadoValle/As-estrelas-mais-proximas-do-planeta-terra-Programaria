###  📝As estrelas mais próximas do Planeta 🌏 !

Projeto autoral final desenvolvido no curso euProgramo, FE#2-T03 da Programaria;

Nesse curso aprendi sobre responvidade com @media e o efeito accordion com HTML5.



### 📌 Objetivo  
Desenvolver uma aplicação web interativa que apresenta informações sobre algumas das estrelas mais próximas do planeta Terra.  
Este projeto foi realizado como parte do **curso de Front-End da [Programaria](https://www.programaria.org/)**, uma iniciativa que incentiva e apoia mulheres na tecnologia. 💖 

---

### 🛠️ Tecnologias Utilizadas  
- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla JS)**  

---

### ⚙️ Funcionalidades  
- Exibição de cards com imagem, nome e descrição de cada estrela  
- Uso de elementos `<details>` e `<summary>` para expandir informações  
- Responsividade para diferentes tamanhos de tela  
- Footer com créditos e links para GitHub, Portfólio e Programaria  

---

### 🧠 Conceitos Aplicados  
- Estruturação semântica com **HTML5**  
- Estilização com **CSS**  
- Manipulação do **DOM** com JavaScript  
- Criação e iteração de objetos em JavaScript  
- Experiência do Usuário (**UX**) com foco em clareza e acessibilidade
